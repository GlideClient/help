SoarClient Discontinuation

04/07/2025
EldoDebug & GlideClient

This article refers to the SoarClient project.

SoarClient development has come to an end.

Eldo has stated that he is sorry to inform the community that SoarClient is no longer in development.
With this the SoarClient 8 source code has been published on github so anyone can modify it or re-release it themselves.
Eldo was thankful for the support the SoarClient community gave him over the development of Soar.
For those who want to continue getting support for SoarClient 7, GlideClient will provide that with GlideClient 7.
It is recomended that people on SoarClient 7 migrate to GlideClient.
Note: GlideClient wont provide support for SoarClient 7, only GlideClient 7.
